# Overview
AI-Gym-Tracker is a real-time computer vision project that analyzes your deadlift form using MediaPipe Pose and OpenCV. It provides immediate feedback, highlights incorrect posture, and counts repetitions â€” making it a handy tool for fitness tracking and injury prevention.

## Key Features
Real-time webcam pose tracking

Calculates hip, knee, and elbow joint angles

Highlights bad posture using red overlay

Counts reps based on hip movement

Gives live feedback on form corrections
